在需要兼容 iOS 7.0 及以下的项目中使用 Alamofire

2015.08.14 EyreFree iOS  热度 ℃

Alamofire 是 iOS 和 OS X 上最受欢迎的第三方库之一，它在 Github 上面获得了 15638 个 stars 和2304 个 forks，是使用最广的开源项目之一。
