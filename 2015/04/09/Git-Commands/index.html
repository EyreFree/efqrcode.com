Git 常用命令备忘

SubModule

1. 获取所有 SubModule

2. 删除某个 SubModule

例如：submodule_xxx

Tag

1. 添加 Tag 并上传本地 Tag 到服务器

例如：2.333

2. 删除本地 Tag 并同步到远程

例如：2.333

Branch

1. 查看所有分支

2. 同步本地与远程分支

删除远程不存在的本地分支

Commit

1. 合并本地的最后两次 Commit

2. 修改上一次的 Commit 信息

3. 撤销所有未提交的本地修改

Remote

1. 删除远程仓库地址

2. 添加远程仓库地址

3. Push 本地分支到指定远程分支

例如：Push 本地当前分支到远程仓库 origin 的 master 分支

Other

1. 设置本地用户名、邮箱

例如：设置用户名为 eyrefree，邮箱为 eyrefree@eyrefree.org
